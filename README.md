# 项目简介

## 核心功能
- 车位信息展示
- 车位租用服务
- 公告管理
- 个人中心
- 在线客服
- 订单管理
- 模拟支付
- 个人中心管理
- 用户管理
- 车位区域管理
- 车位信息管理
- 车位租用管理
- 车位退组管理
- 轮播图管理

## 适用场景
- 停车场运营管理
- 企业内部停车场管理
- 住宅小区停车场管理

## 技术特点
- 使用Python编程语言，基于Django框架
- 前端采用Vue、elementui技术
- 后端数据库使用MySQL
- 系统界面友好，操作简便
- 支持车位信息实时更新
- 订单管理系统提供便捷的支付体验
